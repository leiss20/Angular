Bienvenido a Mi Página HTML

Este es un ejemplo de cómo mostrar texto formateado en una página web utilizando HTML. Puedes incluir títulos, párrafos, listas, énfasis y mucho más.

HTML es el lenguaje base de la web y permite estructurar el contenido de forma sencilla y clara.

  • Fácil de aprender
  • Compatible con todos los navegadores
  • Ideal para crear sitios web estáticos
